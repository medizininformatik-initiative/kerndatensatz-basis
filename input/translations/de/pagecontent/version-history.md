Dieser Abschnitt bietet Informationen zur Versionshistorie des MII-Kerndatensatz-Basis-Implementierungsleitfadens.

### Aktuelle Version

Dies ist Version **2026.0.0-dev.1** des MII-Kerndatensatz-Basis-Implementierungsleitfadens, veröffentlicht als Entwicklungsversion.

### Versionierungsschema

Ab Version 2024.0.0 folgt der MII-Kerndatensatz einem **Calendar Versioning (CalVer)**-Schema:

* **YYYY.MINOR.PATCH**-Format
* **YYYY** = Jahr, für das der IG anwendbar und zur Nutzung vorgesehen ist
* **MINOR** = Minor-Versionsnummer (wird für Feature-Ergänzungen erhöht)
* **PATCH** = Patch-Versionsnummer (wird für Fehlerbehebungen erhöht)
