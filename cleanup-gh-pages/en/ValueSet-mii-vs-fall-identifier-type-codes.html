ValueSet: MII VS Fall Identifier Type Codes

Official URL: https://www.medizininformatik-initiative.de/fhir/core/modul-fall/ValueSet/identifier-type-codes Version: 2026.0.0
Computable Name: MII_VS_Fall_IdentifierTypeCodes

A coded type for an identifier that can be used to determine which identifier to use for a specific purpose.
